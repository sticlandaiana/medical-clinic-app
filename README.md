## Overview
Acest proiect are ca scop dezvoltarea unui sistem software pentru administrarea eficientă a unei clinici medicale. Aplicația va permite gestionarea pacienților, programărilor și altor resurse esențiale.

## Features (în dezvoltare)
- Gestionare pacienți  
- Programări consultații  
- Administrare medici  
- Evidență medicală
